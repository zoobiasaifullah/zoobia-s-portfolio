## üé® Customization

### Updating Personal Information

1. **Hero Section**: Edit the name and description in `index.html`
2. **About Section**: Update the about text to reflect your background
3. **Skills**: Modify the skills grid to showcase your competencies
4. **Projects**: Replace project cards with your own work
5. **Contact Info**: Update email and social media links

### Color Scheme

The color scheme can be customized by modifying CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --text-color: #1f2937;
    --text-light: #6b7280;
}
```

### Adding Project Images

Place your project images in the `assets/images/` directory and update the image sources in `index.html`.

## üåê Deployment

This portfolio can be deployed on various platforms:

### GitHub Pages

1. Push your code to GitHub
2. Go to repository Settings > Pages
3. Select the main branch as source
4. Your site will be published at `https://yourusername.github.io/repository-name`

### Netlify

1. Connect your GitHub repository to Netlify
2. Netlify will automatically build and deploy your site
3. Custom domain configuration is available

### Vercel

1. Import your GitHub repository
2. Vercel will automatically detect the project settings
3. Deploy with zero configuration

## üìù Usage

1. Clone the repository:
   ```bash
   git clone git@github.com:zoobiasaifullah/zoobia-s-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd zoobia-s-portfolio
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

4. Access the site at `http://localhost:8000`
